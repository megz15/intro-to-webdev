What is Web Dev?

Web Development is the process of developing a website or a web application.

It typically involves applying a combination of different technologies called stacks, which include programming languages, databases, frameworks, among other stuff to build and deploy interactive and dynamic websites.

What is a front end and how is it different from a back one?

Frontend

Frontend development involves creating user interfaces of a website or web application while keeping user experience in mind (UI/UX).

This includes designing and developing whatever the client interacts with, including the layout, visual components, and dynamic, interactive features, usually using HTML, CSS, and JavaScript.

Backend

Backend development focuses on the server-side of web development. It involves user authentication and authorization, managing data, APIs among many other things.

Backends are written in programming languages like JavaScript, Python, Dart, Rust, PHP etc.

NodeJS is a commonly used js runtime environment that is used as a backend, Flask and Django are common Python frameworks for web dev.

How do I start making frontend stuff?

To start with frontend, you will need to learn the basics of HTML, CSS and JavaScript.

HTML

HTML, or HyperText Markup Language, is the markup language used to create the structure of a webpage.

HTML consists of a series of elements, or content enclosed in tags.

Here's a simple example:


		<html>
		<head>
			<title>Example</title>
		</head>
		<body>
			<h1>Welcome to the workshop!</h1>
			<p>This is a simple example.</p>
			<ul>
				<li>Item 1</li>
				<li>Item 2</li>
				<li>Item 3</li>
			</ul>
		</body>
		</html>

CSS

CSS, or Cascading Style Sheets, is a style sheet language used to make HTML more sexy.

It involves selectors, properties, and values, as well as how to apply styles to different HTML elements.

Here's a simple example:


		/* Styles for the body of the page */
		body {
		font-family: Arial, sans-serif;
		background-color: coral;
		}

		/* Styles for the heading */
		h1 {
		color: #252525;
		text-align: center;
		}

		/* Styles for the paragraph */
		p {
		font-size: 18px;
		line-height: 1.5;
		margin-bottom: 20px;
		}

		/* Styles for the list */
		ul {
		list-style-type: disc;
		margin-left: 20px;
		}

		/* Styles for the list items */
		li {
		font-size: 16px;
		line-height: 1.2;
		}

CSS Frameworks

These are pre-written sets of CSS and sometimes JavaScript code that provide a set of standardized styles, layouts, and UI components that can be easily customized and reused throughout a project.

Well known CSS frameworks:

  • Bulma
  • Bootstrap
  • Tailwind
  • Foundation
  • Skeleton
  • Materialize
  • PureCSS

Bootstrap

Directed at responsive, mobile-first front-end web development, Bootstrap focuses on simplifying the development of web pages with its several inbuilt HTML, CSS and JavaScript components.

Bulma

Bulma is designed to be lightweight and easy to use, with a simple and intuitive class names and a variety of easily customizable pre-built CSS components.

Tailwind

Tailwind CSS is designed to be highly customizable, in that allows developers to quickly and easily apply pre-built CSS classes to their HTML elements.

It includes a wide range of pre-built utility classes for typography, spacing, layout, colors, borders, and more.

JavaScript

JS is a programming language used to create interactive and dynamic web pages.

It includes covering a range of programming fundamentals like data types, variables, functions, conditional statements, and loops.

Here's a simple example:


		function addNumbers(num1, num2) {
			var sum = num1 + num2;
			return sum;
		}

		function changeText() {
			var heading = document.getElementById("heading");
			heading.innerHTML = `5 + 15 = ${addNumbers(5, 15)}`;
		}

	changeText();

JavaScript Frameworks

These are pre-written JS code that provide a set of libraries and tools for building web applications.

They allow developers to focus on the high-level aspects of the application, without having to worry about the underlying details and re-inventing the wheel by rewriting trivial code.

Well known JS frameworks:

  • Svelte
  • Vue.js
  • React
  • AngularJS
  • Ember.js
  • Alpine.js

Svelte

Svelte is a modern web framework that focuses on building highly efficient and fast web applications by moving much of the work that typically happens at runtime to build time.

Svelte compiles application code into highly efficient vanilla JavaScript, resulting in smaller bundle sizes, faster load times, and better overall performance.

Vue.js

Vue.JS is a progressive framework that is designed to be incrementally adoptable. It also has a small learning curve, making it an ideal choice for beginners.

Vue.js uses a reactive data binding system. This makes it easy to build dynamic and interactive user interfaces without having to write a lot of boilerplate code.

React

Technically not a framework, React is a popular open-source library developed by Facebook.

One of its main features is its component-based architecture, which allows the interface to be broken down into smaller, reusable pieces of code.

React uses a virtual DOM to update the UI to minimize the number of updates that need to be made to the actual DOM.

How do I create a backend?

Now that you know how to go about making a decent frontend, lets talk about the backend

Do I even need a backend?

A simple static website or even a dynamic one that doesn't need to rely on complex functionality like data storage and user authentication would not need a backend.

However, a backend is necessary when the web application requires dynamic functionality like authentication, user-generated content, or real-time updates.
